# 贡献指南 (Contributing Guide)

感谢你对 G-web 感兴趣！👏 无论是修复 Bug、改进 UI 还是增加新功能，我都非常欢迎。

### 如何参与贡献
1. **Fork** 本仓库到你的账号下。
2. **Clone** 你的 Fork 仓库到本地。
3. 创建一个新的 **Feature 分支** (`git checkout -b feature/AmazingFeature`)。
4. **Commit** 你的修改（请保持 commit 信息清晰）。
5. **Push** 到该分支。
6. 在本仓库提交一个 **Pull Request (PR)**。

### 建议与反馈
如果你发现了 Bug 或者有好的想法，欢迎直接提交 [Issue](../../issues)。

### 开发说明
* 本项目是一个浏览器扩展，开发时请开启 Chrome/Edge 的“开发者模式”。
